# 📊 Data Quality and Exploratory Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing data quality issues and performing basic exploratory data analysis (EDA) using Python and pandas. The dataset is examined for missing values, ZIP code distribution, and date range coverage to understand its structure and completeness.

---

## 🎯 Objectives
- Identify and analyze missing values in the dataset
- Perform ZIP code-level analysis
- Understand dataset time coverage (date range)
- Assess data completeness before further analysis or modeling

---

## 🧰 Tools & Technologies Used
- Python 🐍
- Pandas
- Google Colab
- Jupyter Notebook (optional environment)

---

## 🔍 Data Quality Analysis Performed

### 1. Missing Values Analysis
- Calculated total missing values per column
- Computed percentage of missing data
- Sorted columns by highest missing values
- Identified critical gaps in dataset completeness

---

### 2. ZIP Code Analysis
- Counted missing ZIP code entries
- Calculated number of unique ZIP codes
- Explored geographic data distribution (via ZIP codes)

---

### 3. Date Range Analysis
- Converted date column to datetime format
- Identified earliest and latest dates in dataset
- Extracted range of years covered in the data
- Checked temporal spread of records

---

## 📊 Key Insights
- Some columns contain missing values that may require cleaning or imputation
- ZIP code data helps understand geographic coverage and completeness
- Dataset spans multiple years, providing a time-based perspective for analysis
- Data quality checks are essential before performing deeper analysis or modeling
